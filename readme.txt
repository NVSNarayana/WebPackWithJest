1. webpack is open source module bundler for javascript
2. babel is open source javascript transcompiler, 
used to convert ECMAScript 2015+ code into compatable version of javascript 
that can be run by older javascript engine.
3. In this project we are not using bable, babel-loader for transcompilation. We are using ts-loader, typescript

4. npm run build -- to build
5. npm run serve
6. npm test
7. npm run test:coverage
